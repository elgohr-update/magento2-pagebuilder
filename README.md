## Magento Pagebuilder for Magento 2 Integration

This project is the Magento 2 Pagebuilder module for VueStorefront Magento 2 Integration.

This module is being developed by superheroes from [Caravel](https://github.com/caravelx) ❤️

## How to start if you want to try out the integration

```
yarn global add @vue-storefront/cli
```
```
vsf init <project_name> && cd <project_name> && yarn && yarn dev
```

## How to start if you want to contribute?

Want to contribute? Ping us on `magento2-vsf2` channel on [our Discord](http://discord.vuestorefront.io)!

### Requirements:
- NodeJS v16 or later
- Yarn (npm is not supprted yet)
- Magento >= v2.4.3 instance for GraphQL endpoint
